# Wida Putri - Portfolio Website

A beautiful, responsive portfolio website built with Next.js 16, featuring elegant animations and modern design based on Figma specifications.

## ‚ú® Features

- **Responsive Design**: Fully responsive layout that adapts seamlessly from mobile to desktop
- **Smooth Animations**: Powered by Framer Motion for fluid, engaging microanimations
- **Modern Typography**: Custom Google Fonts (Caveat, Instrument Sans, Bricolage Grotesque)
- **Gradient Background**: Dynamic gradient mesh background with animated elements
- **Interactive Elements**: Hover effects and spring animations on all interactive components
- **Optimized Performance**: Built with Next.js 16 for optimal performance and SEO
